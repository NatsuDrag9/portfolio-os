# portfolio-os

A portfolio mimicking Windows 11 with Fluent design system

## Getting Started

Clone the repo

```
git clone https://github.com/NatsuDrag9/portfolio-os
```

Run the app

```
cd portfolio-os

yarn install

yarn dev

```

## System Requirements

| Technology | Version  |
| ---------- | -------- |
| Node.js    | >= 18.12 |
| React      | ^19.2.0  |
| TypeScript | ~5.9.3   |
| Vite       | ^6.0.0   |
| Storybook  | 8.6.14   |
