# Getting Started

## Installation

To get started with Storybook, follow these simple steps:

1. Clone the Repository

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project root

```bash
   cd p4workflow
```

3. Install all packages / dependencies

```bash
   yarn install
```

4. Run storybook

```bash
   yarn storybook
```
